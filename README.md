# 🧴 Cosmore

AI-powered cosmetic ingredient scanner built using OCR, computer vision and fuzzy matching.

## Features

- 📷 Scan cosmetic labels from images
- 📸 Capture labels directly using your phone camera
- 📝 Paste ingredient lists directly
- 🔍 OCR-powered ingredient extraction
- 🧠 Fuzzy matching against 26,000+ cosmetic ingredients
- 🌿 Ingredient family classification
- ⭐ Ingredient quality ratings
- 📊 Product health summary
- 📈 Ingredient family distribution
- 📥 Download results as CSV
